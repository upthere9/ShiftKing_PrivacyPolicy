# Política de Privacidade

**9&U Inc.** (doravante denominada “Empresa”) valoriza a privacidade dos usuários do aplicativo **ShiftKing** (doravante denominado “Aplicativo”) e trata dados pessoais de acordo com as leis e regulamentos aplicáveis.

Esta Política de Privacidade se aplica ao aplicativo móvel ShiftKing e aos serviços relacionados.

---

## 1. Finalidade do tratamento de dados pessoais

A Empresa trata dados pessoais para as seguintes finalidades:

- Fornecer e manter os recursos e serviços do Aplicativo.
- Analisar erros e melhorar a qualidade do serviço.
- Monitorar e melhorar o desempenho do Aplicativo.
- Exibir anúncios e medir seu desempenho.
- Responder a dúvidas dos usuários e prestar suporte.
- Revisar escalas de trabalho enviadas voluntariamente pelos usuários para melhorar o banco de dados de escalas padrão.

A Empresa não utiliza dados pessoais para finalidades diferentes das descritas acima.

---

## 2. Dados que coletamos

### 2.1 Dados que não coletamos diretamente

O ShiftKing não exige cadastro nem login.

A Empresa não coleta diretamente, por meio do Aplicativo, dados pessoais como nome, número de telefone ou endereço de e-mail do usuário.

### 2.2 Dados fornecidos voluntariamente pelo usuário

Os seguintes dados são enviados à Empresa somente quando o usuário opta por fornecê-los voluntariamente:

- Conteúdo incluído em e-mails de suporte.
- Arquivos anexados a e-mails de suporte.
- Escalas de trabalho enviadas por meio da função de envio de escalas.

As escalas enviadas são revisadas pelo desenvolvedor exclusivamente com a finalidade de melhorar o banco de dados de escalas padrão.

### 2.3 Dados que podem ser coletados automaticamente

Para fornecer, manter e melhorar o Aplicativo, os seguintes dados podem ser tratados automaticamente:

- Modelo do dispositivo.
- Versão do iOS.
- Versão do Aplicativo.
- Informações de uso do Aplicativo.
- Informações de falhas e diagnóstico.
- Informações de desempenho.
- Identificador de publicidade (Advertising Identifier, IDFA) e informações relacionadas a anúncios, quando aplicável e, se necessário, após o consentimento do usuário.

Esses dados podem ser tratados por meio do Google Firebase, Google AdMob e serviços de mediação de anúncios.

---

## 3. Serviços de terceiros

A Empresa não vende os dados pessoais dos usuários nem os divulga arbitrariamente a terceiros.

No entanto, o Aplicativo utiliza os seguintes serviços de terceiros, que podem tratar determinados dados na medida necessária para fornecer seus respectivos serviços:

- Google Firebase
- Google AdMob
- Meta Audience Network
- Unity Ads

Cada um desses serviços trata os dados de acordo com sua própria política de privacidade.

---

## 4. Prestadores de serviços e serviços utilizados

O Aplicativo utiliza os seguintes serviços para apoiar sua operação e melhorar sua qualidade:

- Firebase Analytics
- Firebase Crashlytics
- Firebase Cloud Messaging (FCM)
- Firebase Remote Config
- Firebase Performance Monitoring
- Google AdMob
- Meta Audience Network
- Unity Ads

Esses serviços podem tratar os dados necessários para fornecer suas respectivas funcionalidades.

---

## 5. Permissões utilizadas pelo Aplicativo

O ShiftKing pode solicitar as seguintes permissões do sistema.

### Calendário

O acesso de leitura e gravação ao calendário é utilizado para permitir que os usuários adicionem, consultem ou gerenciem seus horários de trabalho e escalas no calendário.

### Notificações

A permissão de notificações é utilizada para fornecer lembretes locais relacionados a horários de trabalho, escalas e outros recursos do Aplicativo.

### iCloud

Se o usuário ativar essa função, o iCloud poderá ser utilizado para fazer backup e restaurar os dados do Aplicativo.

A Empresa não utiliza essas permissões para finalidades diferentes das descritas acima.

---

## 6. Envio de escalas de trabalho

O Aplicativo inclui uma função opcional para envio de escalas de trabalho.

As escalas são transmitidas à Empresa somente quando o usuário escolhe expressamente enviá-las.

As escalas enviadas são utilizadas exclusivamente para revisão e para melhorar o banco de dados de escalas padrão.

As escalas pessoais que não forem enviadas voluntariamente permanecem armazenadas apenas no dispositivo do usuário e não são transmitidas aos servidores da Empresa.

---

## 7. Retenção de dados pessoais

A Empresa mantém os dados pessoais apenas pelo tempo necessário para cumprir as finalidades descritas nesta Política de Privacidade ou quando exigido pela legislação aplicável.

As informações fornecidas por e-mail de suporte são mantidas apenas pelo período necessário para resolver a solicitação.

As escalas enviadas voluntariamente pelos usuários podem ser mantidas pelo período necessário para sua revisão e para a melhoria do banco de dados de escalas padrão.

---

## 8. Proteção de dados pessoais

A Empresa adota medidas técnicas e organizacionais adequadas para proteger os dados pessoais, incluindo:

- Restrição de acesso aos dados pessoais.
- Aplicação de criptografia quando apropriado.
- Uso de tecnologias de segurança atualizadas.
- Limitação e gerenciamento do pessoal com acesso a dados pessoais.

---

- Acesso aos seus dados pessoais.

- Limitação do tratamento de dados pessoais.
